Tests in this folder are divided into End-to-End (e2e) and unit.

End-to-End (e2e) Tests
=======================
- End-to-End testing is performed with tools from webdriver.io and selenium


Unit testing
==============
- Download and make executable PHPUnit to ./bin/php/phpunit or setup PHPunit.phar in your path
- run the unit tests with $ ./bin/php/phpunit ./test/unit/run.php
