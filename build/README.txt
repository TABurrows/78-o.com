The build directory is the place for building and deploying 78-o.com

These build\server\vagrant\* files are required by the vagrant up call issued from the root directory:
- The build\server\vagrant\install.sh is install script for local vagrant versions of the site
- The server\vagrant\data directory contains the test data from a sqlite3 dataset for testing local instances

The build\client\* files are required by cordova build process.
